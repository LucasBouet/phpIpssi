# Blog PHP - Partie 1

## Description

Projet de blog développé en PHP avec architecture de base.

## Fonctionnalités

- Liste des articles
- Page article individuel
- Header/Footer réutilisables
- Fonctions utilitaires
- Design responsive

## Installation

1. Placer le dossier dans `htdocs`
2. Accéder à `http://localhost/blog/`

## Structure

- `index.php` : Page d'accueil
- `article.php` : Page article
- `includes/` : Header, footer, functions
- `data/` : Données des articles
- `assets/` : CSS et ressources
